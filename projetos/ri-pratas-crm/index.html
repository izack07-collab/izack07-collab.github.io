Sistema de Operações Digitais (SOD)

React (Vite) TypeScript Shadcn/UI Supabase n8n

O "Cérebro" operacional da RI Pratas. Uma plataforma Backoffice centralizada que substitui planilhas e integra estoque, vendas e logística em tempo real, servindo como fonte da verdade para o E-commerce e Agentes de IA.

Principais Funcionalidades

  • Motor Operacional: Automatiza tarefas repetitivas. Vendas atualizam estoque, calculam frete e enriquecem o CRM automaticamente.
  • Comando de Estoque: Gestão de variações (tamanhos) em tempo real, impedindo vendas sem estoque pelo Agente de IA.
  • Customer 360: Perfil completo do cliente com cálculo automático de LTV (Lifetime Value) e histórico de compras.
  • Kanban de Pedidos: Pipeline visual drag-and-drop para a equipe de logística (Pago -> Separação -> Enviado).

Tecnologia

  • Frontend reativo com React + Vite.
  • Design System "Minimal Luxury" com Shadcn/UI.
  • Orquestração de eventos com n8n.
  • Banco de dados relacional PostgreSQL.
